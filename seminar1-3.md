## Начало работы с GIT 

*в начале работы не обходимо нициализировать рабочий файл* 

**git init**

*после инециализации файла необходимо его запутить в работу*

**git add название файла** 

*после запуска в работу необходимо создать коммид* 

**git commid -m**

**git commid -a**

**git commid -am**
